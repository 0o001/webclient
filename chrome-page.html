MEGA Chrome Extension

This app will allow you to install MEGA into your browser for enhanced security and boot MEGA securely from your computer rather than from our servers.
Any MEGA URL will be captured by this app and stay local (no JavaScript will be loaded from our servers). Secure (auto-) updates are provided thanks to cryptographic private key signing.

1
Download MEGA for Chrome
2
Open the extensions page in Chrome
(menu> tools>extensions) or type
chrome://extensions in your address bar.
3
Drag and drop the downloaded file MEGA.crx file to the Chrome extensions page and confirm installation.
